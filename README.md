# Experiment-UI

>这是我用Qt5.14.1 64位写的实验上位机，其中包含登录界面，菜单界面，菜单界面下可以选择不同的按钮进入不同的任务，包括静息任务、按键任务、最大力测试、捏力任务和串口设置。
>本项目的核心是串口设置，连接压力传感器后，可以通过RS485读取串口数据
